Sobre o Projeto¶

Crianex é um software SaaS de gestão e portfólio de projetos, desenvolvido como projeto da disciplina de Requisitos de Software (REQ-2026.1-T02).

Informação Detalhe
Disciplina Requisitos de Software — 2026.1
Turma T02
Produto Plataforma SaaS de Gestão de Projetos

Objetivo do Projeto¶

O projeto Crianex é desenvolvido no contexto da disciplina de Requisitos de Software (REQ — 2026.1) da Universidade de Brasília (UnB), com o objetivo de aplicar, na prática, as técnicas e processos de Engenharia de Requisitos sobre um produto de software real.

A disciplina tem como propósito capacitar os alunos a:

  • Elicitar requisitos junto a stakeholders reais, utilizando técnicas como entrevistas, Rich Picture e Diagrama de Ishikawa;
  • Analisar e especificar requisitos de forma estruturada, por meio de user stories com critérios de aceite;
  • Priorizar requisitos utilizando técnicas como MoSCoW, alinhadas ao valor de negócio do produto;
  • Verificar e validar requisitos ao longo do ciclo de desenvolvimento iterativo;
  • Gerenciar mudanças de requisitos de forma rastreável e documentada;
  • Produzir artefatos de documentação profissionais, publicados e versionados via GitHub Pages.

O produto escolhido pela equipe é a Crianex Hub, uma plataforma SaaS voltada para gestão interna e divulgação de portfólio de projetos de uma Software House B2B.
