# Evidências do Projeto

:::tip[Como navegar]
As **features entregues** têm uma página por feature (com critérios BDD, regras de negócio e screenshots por requisito). Os **artefatos técnicos** (arquitetura, color modeling, feature cards, protótipo) estão linkados nas tabelas abaixo.
:::
