# Entrega — Unidade 3


## Contexto

A Unidade 3 cobriu a execução completa da IT1 — do início do desenvolvimento das features (a partir das issues criadas no TDR) até o encerramento formal com Formal Client Validation aprovada pelo CEO. **Período:** 19/05/2026 – 16/06/2026
