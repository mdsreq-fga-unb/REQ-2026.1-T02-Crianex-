## Priorização: Valor × Esforço

```
IP = VB / PT     onde PT = (CX + ES) / 2

IP ≥ 1,50 → Alta prioridade (Q1)
IP 1,00–1,49 → Média prioridade (Q2)
IP < 1,00 → Baixa prioridade (Q3/Q4)
```

| Sigla | Descrição | Escala |
|-------|-----------|--------|
| **VB** | Valor de Negócio | 1–5 |
| **CX** | Complexidade técnica | 1–5 |
| **ES** | Esforço de implementação | 1–5 |
| **PT** | Peso técnico = (CX + ES) / 2 | — |
| **IP** | Índice de Prioridade = VB / PT | — |
