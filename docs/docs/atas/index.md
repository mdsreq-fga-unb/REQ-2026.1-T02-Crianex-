# Atas de Reunião

Registro cronológico de todas as reuniões realizadas pela equipe Crianex.

---

!!! info "Como registrar uma ata"
    Crie um novo arquivo no formato `YYYY-MM-DD.md` dentro desta pasta e adicione-o à navegação no `mkdocs.yml`.

## Modelo de Ata

Copie o template abaixo ao criar uma nova ata:

```markdown
# Reunião — DD/MM/AAAA

**Data:** DD/MM/AAAA
**Horário:** HH:MM – HH:MM
**Local:** (Presencial / Google Meet / Discord)
**Facilitador:** Nome
**Redator:** Nome

## Participantes

| Nome | Papel |
|------|-------|
| ...  | ...   |

## Pauta

1. Item 1
2. Item 2

## Discussões e Decisões

### Item 1 — Título
...

## Encaminhamentos

| # | Tarefa | Responsável | Prazo |
|---|--------|-------------|-------|
| 1 | ...    | ...         | ...   |

## Próxima Reunião

**Data prevista:** DD/MM/AAAA
```
